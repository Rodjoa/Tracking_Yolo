
## Historial de trayectoria

Variable:

```
self.track_history
```


self.track_history[n] corresponde al historial de trayectorias del objeto con id n.

Estructura:

```
ID -> lista de coordenadas del centroide de la caja del objeto
```

Ejemplo:

```
{
23:[
(450,200),
(455,205),
(460,210)
]
}
```
