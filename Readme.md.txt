# Sistema de Conteo Vehicular y Peatonal mediante YOLO

Proyecto desarrollado durante una práctica profesional.

## Características

- Detección mediante YOLO.
- Tracking de objetos.
- Conteo mediante líneas virtuales.
- Detección de trayectorias vehiculares mediante múltiples líneas.
- Conteo peatonal mediante una línea.
- Documentación técnica en Obsidian.
